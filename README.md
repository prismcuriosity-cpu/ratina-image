# ratina-image
## Summary

- Adds `Unsupervised_Retinal_Vessel_Segmentation.ipynb` — a fully unsupervised, end-to-end pipeline for retinal vessel segmentation on the **FD3611** dataset (5 categories: Diabetic Retinopathy, Media Hazy, Myopic Retinopathy, Normal, Optic Disc Disorder)
- No ground-truth labels required at any stage
- Optimised for **Google Colab T4 GPU** (mixed-precision AMP, batch size tuned to 15 GB VRAM)

## Architecture

**Self-Supervised Attention U-Net + ASPP Bottleneck**

| Component | Detail |
|---|---|
| Encoder | 4-stage double-conv blocks with MaxPool |
| Bottleneck | Atrous Spatial Pyramid Pooling (dilations 6/12/18 + GAP) |
| Decoder | Transposed-conv upsampling with soft Attention Gates |
| Head 1 | Vessel probability map (sigmoid) |
| Head 2 | Image reconstruction head (self-supervised auxiliary task) |
| Parameters | ~31 M trainable |

## Training Pipeline

**Phase 1 — Pseudo-label training (30 epochs)**
- Pseudo-labels generated via Frangi + Sato vesselness filter ensemble (no annotations)
- Loss: Weighted BCE + Dice + Contrastive margin loss
- Optimizer: AdamW + CosineAnnealingWarmRestarts

**Phase 2 — Self-training refinement (20 epochs)**
- Model predictions (60%) ensembled with original Frangi labels (40%) → refined pseudo-labels
- Fine-tuned at 10× lower LR with CosineAnnealingLR
